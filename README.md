# Key Value DB

A Key-Value store database built in Python that utilizes a Linked List implementation and loads from disk on crash or program restart.

## How to run in your envoirnment

1. First, clone the repo using: `git clone git@github.com:kyibell/key-value-store-db.git`
2. For gradebot, run: `./gradebot --run "python3 main.py"`
   - For just a regular terminal, just run: `python3 main.py`

Both of these steps allow you to start putting commands towards the server!
